One-Time Setup

Paste your Google Apps Script URL below to connect your playbook to Google Sheets. Your data will save automatically after every transaction.

  1. Go to drive.google.com and create a new Google Sheet
  2. Name it Phil's Playbook Data
  3. Inside the sheet, create two tabs: State and Log
  4. Click Extensions → Apps Script
  5. Delete everything in the editor, paste in the script from the setup guide, click Save
  6. Click Deploy → New deployment → Web app
  7. Set "Who has access" to Anyone, click Deploy
  8. Copy the Web App URL and paste it below
